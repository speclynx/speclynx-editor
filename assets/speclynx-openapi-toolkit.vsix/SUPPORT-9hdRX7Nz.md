# Support

## How to file issues and get help

This project uses [GitHub Issues](https://github.com/speclynx/vscode-openapi-toolkit/issues) to track bugs and feature requests. Please search the existing
issues before filing new issues to avoid duplicates. For new issues, file your bug or feature request as a new Issue.

For help and questions about using SpecLynx OpenAPI Toolkit, please use [SpecLynx GitHub Discussions](https://github.com/orgs/speclynx/discussions).

You can also reach out to us via [info@speclynx.com](mailto:info@speclync.com) email.
